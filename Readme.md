# 📋 Aura Task Manager

Aura Task Manager is a clean, modern, and responsive **Project Management Tool** built using **HTML**, **CSS**, and **Vanilla JavaScript**. It helps users organize tasks efficiently with features like task creation, editing, deletion, searching, filtering, and LocalStorage support.

---

## 🚀 Features

- Modern & Minimal UI
- Fully Responsive Design
- Dashboard Statistics
- Add New Task
- Edit Existing Task
- Delete Task
- Search Tasks
- Filter by Task Status
- Priority Levels (High, Medium, Low)
- Due Date Management
- LocalStorage Support
- Success Notifications
- Clean Card-Based Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
- Font Awesome
- Google Fonts (Inter)

1. Download or clone this repository.


2. Open the project folder.

3. Double-click **index.html** or open it in any modern web browser.

No installation, build tools, or server is required.
